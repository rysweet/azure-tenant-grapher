# Microsoft Learn Resources for Azure Administration

Curated learning paths and resources from Microsoft Learn for Azure administrators.

## Learning Paths

### Azure Administrator (AZ-104)

**Prerequisites and Fundamentals:**

- Azure Fundamentals (AZ-900)
  - https://learn.microsoft.com/training/paths/az-900-describe-cloud-concepts/
  - https://learn.microsoft.com/training/paths/az-900-describe-azure-architecture-services/
  - https://learn.microsoft.com/training/paths/az-900-describe-azure-management-governance/

**AZ-104 Learning Paths:**

1. **Manage identities and governance** (15-20 hours)
   - Configure Azure Active Directory (Entra ID)
   - Manage users and groups
   - Configure role-based access control (RBAC)
   - Manage subscriptions and governance
   - https://learn.microsoft.com/training/paths/az-104-manage-identities-governance/

2. **Implement and manage storage** (10-15 hours)
   - Configure storage accounts
   - Configure blob storage
   - Configure Azure files and Azure File Sync
   - https://learn.microsoft.com/training/paths/az-104-manage-storage/

3. **Deploy and manage Azure compute resources** (15-20 hours)
   - Configure virtual machines
   - Automate deployment with ARM templates
   - Create and configure containers
   - Configure Azure App Service
   - https://learn.microsoft.com/training/paths/az-104-manage-compute-resources/

4. **Configure and manage virtual networking** (15-20 hours)
   - Configure virtual networks
   - Configure name resolution
   - Secure access to virtual networks
   - Configure load balancing
   - https://learn.microsoft.com/training/paths/az-104-manage-virtual-networks/

5. **Monitor and maintain Azure resources** (10-15 hours)
   - Monitor resources by using Azure Monitor
   - Implement backup and recovery
   - https://learn.microsoft.com/training/paths/az-104-monitor-backup-resources/

## Learning Path Recommendations by Role

### Cloud Administrator (Entry Level)

1. Start with AZ-900 (Azure Fundamentals)
2. Progress to AZ-104 (Azure Administrator)
3. Practice with sandbox environments
4. Gain 6-12 months hands-on experience

### DevOps Engineer

1. Complete AZ-104 (foundation)
2. Focus on IaC modules (Bicep, ARM templates)
3. Complete AZ-400 (DevOps Engineer)
4. Practice CI/CD pipeline creation

### Security Specialist

1. Complete AZ-104 (foundation)
2. Focus on identity and RBAC modules
3. Complete AZ-500 (Security Engineer)
4. Learn Azure Policy and compliance tools

### Solutions Architect

1. Complete AZ-104 (foundation)
2. Gain multi-service experience
3. Complete AZ-305 (Solutions Architect)
4. Study architecture patterns and case studies

## Study Tips

1. **Hands-On Practice**: Use free Azure credits and sandbox environments
2. **Real-World Projects**: Apply learning to actual scenarios
3. **Join Study Groups**: Microsoft Learn community forums
4. **Regular Review**: Revisit modules and documentation regularly
5. **Build Lab Environment**: Create personal dev/test environment
6. **Document Learnings**: Keep notes and create personal knowledge base
7. **Follow Azure Updates**: Stay current with service changes
8. **Practice Exams**: Use official practice assessments before certification
