# Learning Path Builder

Creates personalized learning paths for technologies, frameworks, or concepts.

## When to Use

- Learning a new technology or framework
- Onboarding team members
- Hackathon preparation
- Career development planning
- Filling knowledge gaps

## How to Use

Simply tell Claude what you want to learn:

```
I want to learn React and I have 10 hours per week. I know vanilla JavaScript.
```

The skill will:

1. Assess your current level
2. Define clear goals
3. Create a sequenced learning path
4. Curate high-quality resources
5. Set up progress tracking

## Example Learning Paths

See [examples/](./examples/) for complete learning path templates.

## Features

- **Personalized**: Adapts to your skill level and learning style
- **Progressive**: Concepts sequenced from simple to complex
- **Resource-Rich**: Curates docs, tutorials, exercises, and projects
- **Trackable**: Built-in progress tracking and milestones
- **Flexible**: Adjusts based on your feedback

## Philosophy

Follows amplihack's principles:

- **Ruthless Simplicity**: Clear, actionable steps
- **Zero-BS**: All resources are real and accessible
- **Modular**: Each learning module is self-contained
