## Summary of UI States

| State | Components Visible | User Actions Available |
|-------|-------------------|------------------------|
| **Idle** | Mode selector, Config panel | Switch modes, fill form, preview, execute |
| **Previewing** | Config panel, Preview results | View estimates, back to config, proceed |
| **Running** | Progress monitor, Live logs | Monitor progress, stop operation |
| **Success** | Results panel, Statistics | View details, run another, visualize |
| **Error** | Error panel, Recovery options | View errors, retry, clean, back |
| **Statistics** | Stats modal overlay | View details, export, refresh, close |

## Color Scheme

- **Primary**: #2196F3 (Blue) - Actions, progress
- **Success**: #4CAF50 (Green) - Completion, validation passed
- **Error**: #F44336 (Red) - Failures, critical errors
- **Warning**: #FF9800 (Orange) - Warnings, synthetic nodes
- **Info**: #00BCD4 (Cyan) - Information messages
- **Neutral**: #757575 (Grey) - Disabled states, secondary text

## Typography

- **Headings**: Bold, 16-18px
- **Body**: Regular, 14px
- **Captions**: Regular, 12px
- **Code/Logs**: Monospace, 13px
- **Statistics**: Bold, 20-24px (for numbers)

## Spacing

- **Panel padding**: 24px
- **Section spacing**: 16px
- **Element spacing**: 8px
- **Compact spacing**: 4px
- **Card padding**: 12-16px
